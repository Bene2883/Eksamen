GRUNDLÆGGENDE ANIMATION

Temaet du kan læse om her, handler om vores eget spil som vi skulle kode og designe selv. Designet af spillet er lavet i illustrator. I dette tema blev vi også præsenteret for Javascript for første gang.


For at forberede os til at kode spillet gjorde vi brug af idegenerering og aktivitetsdiagrammer. Vi tegnede vores spil-elementer og figur i illustrator og fik skabt os et univers som vi kunne bygge vores spil udfra.


I temaet blev vi også introduceret til den rette måde at lave lydoptagelser på, så vores spil kunne vækkes mere til live og gøres mere spænende og sjovt. vi lærte meget om Javascript i dette tema, som til at starte med var lidt svlrt at sluge. Men som kun er sjovt at sidde og lege med nu.


Derudover gjorde vi også brug af events, if-sætninger, this og random i Javascript.
